01 Executive Summary

Xây dựng một ứng dụng web hiệu năng cao nhằm tự động hóa quy trình hậu cần (logistics) trong ngành Nghiên cứu thị trường. Công cụ giải quyết bài toán xử lý hàng ngàn dòng dữ liệu khảo sát (Top-up data) để phân phối quà tặng (evouchers) và quản lý chi phí đối tác cung cấp mẫu.

02 The Problem

  • • Mất thời gian: Nhân viên phải lọc và tách file Excel hàng giờ đồng hồ cho mỗi dự án.
  • • Rủi ro sai sót: Việc copy-paste thủ công dễ dẫn đến sai số trong tiền thưởng hoặc nhầm thông tin người nhận.
  • • Thiếu minh bạch: Khó khăn trong theo dõi ngân sách thực tế so với dự kiến, nhất là khi làm việc với nhiều vendor đa tỉ giá.

03 The Solution

  • • Automated Splitting: Tự động tách file theo Vendor/Referrer bằng thao tác kéo thả.
  • • Real-time Cost Engine: Tính toán chi phí đa ngoại tệ (USD/VND) và phí dịch vụ.
  • • Budget Forecasting: Ước tính chi phí tích lũy để tối ưu hóa phân bổ mẫu cho các nguồn còn thiếu.

04 Technical Excellence

Modern Stack

Next.js 15, React 19, Tailwind CSS 4 cho hiệu năng và UI cao cấp.

Data Processing

XLSX & JSZip xử lý dữ liệu lớn trực tiếp trên trình duyệt (Client-side).

Visualization

Recharts trực quan hóa Dashboard phân tích ngân sách chuyên sâu.
